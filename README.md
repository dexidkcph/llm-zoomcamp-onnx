# LLM Zoomcamp – ONNX Embedder

This project demonstrates how to generate sentence embeddings using ONNX Runtime instead of the full `sentence-transformers` library.

## Features

- ONNX Runtime inference
- MiniLM sentence embeddings
- Batch embedding generation
- Semantic similarity search
- FAQ dataset vector search

## Tech Stack

- Python
- ONNX Runtime
- Hugging Face Tokenizers
- NumPy
- Jupyter Notebook

## Based on

DataTalksClub LLM Zoomcamp – Module 2, Lesson 9 (ONNX Embedder)
